Definition

A piecewise function is a function defined by subfunctions that apply over different subdomains of the function depending on some corresponding conditions.

A piecewise function can be defined by as many sub functions as is necessary but each sub functions must have some condition that is met for it to occur.

Discontinuities

Discontinuities are points in a function that are non-continuous or where when graphed the function cannot be drawn without segmenting the graphâ€™s line in some way.

Differentiability

Piecewise functions are generally non-differentiable. This is because they are composed of numerous functions that do not have the same derivatives at their intersections. The corresponding one-sided derivatives of any two neighbroing functions must coincide.
