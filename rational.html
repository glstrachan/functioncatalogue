Definition

A rational function is any such function that can be expressed in the form f(x) = g(x) / h(x), where g(x) and h(x) are polynomial.

Non-Permissible Values

Non-permissible values for rational functions are any value of x that results in a denominator of 0.

For the function f(x) = 3x / 2(x - 2), the value of x = 2 will make the f(x) = 6 / 2(0) which is undefined.

Horizontal Asymptotes

Horizontal asymptotes are determined by the degree of the polynomial.

If the degree of the denominator function is greater, then there is a horizontal asymptote at y = 0.

If the degree of the numerator function is larger than there is no horizontal asymptote.

If the degree of the two functions are equal then there is a horizontal asymptote at the division of the coefficients for the highest degree in the respective polynomial functions in numerator and denominator.
